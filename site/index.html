Methodology

Data comes from the USAJobs API, which I started polling in mid-2025. The dataset contains all 2210 (IT Management) series postings that appeared in the current_jobs snapshot from that point forward. Education classifications used GPT-5.4-mini with structured output, verified by GPT-5.4 on edge cases. Skills were extracted from qualification summaries using GPT-5.4-mini. Code and data are on GitHub.

no_education: No degree required at any grade level. Includes postings where education can substitute for experience — since you can qualify with experience alone, there is no education requirement.

education_required: A bachelor's degree or higher is explicitly required with no experience-only alternative.

education_required_higher: Degree required only at higher grade levels, not lower ones.
