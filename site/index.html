Methodology

Data source: All postings come from USAJobs. Recent postings (mid-2025 onward) were pulled directly from the USAJobs API, which includes structured Education and Qualifications fields. Older postings (2023–early 2025) are no longer in the API, so we scraped their announcement pages from usajobs.gov and extracted the Education and Qualifications sections from the HTML. The extraction was validated against API ground truth for overlapping postings. See GitHub for code and data.

Classification: Each posting's Education and Qualifications text was classified by GPT-5.4-mini using structured output (the model must pick from the categories below and provide a verbatim quote from the posting). Classifications were validated against a hand-labeled test set of 22 postings across all categories.

Categories

Experience Only: Education is not a qualifying path at any grade level. You must have experience — a degree alone will not get you in. This includes postings that explicitly say "no substitution of education for experience" as well as postings where only experience is listed as a way to qualify.

Degree is One Path: Education is not mandatory, but the posting explicitly offers it as one way to qualify alongside experience. For example, many 2210 postings under OPM Alternative A say you can qualify with IT-related experience or a 4-year degree in computer science, information systems, etc. If education works at some grade levels but not others, the posting is still classified here — education is a path at at least one grade.

Degree Required: A bachelor's degree or higher is explicitly required with no experience-only alternative. The posting makes clear you cannot qualify without the degree.
