Methodology

Data comes from USAJobs. Current postings are pulled from the API; historical postings (2023–2024) are backfilled by scraping announcement pages. Education classifications use GPT-5.4-mini with structured output via the OpenAI Batch API, verified by GPT-5.4 on edge cases. Code and data are on GitHub.

no_education: Education is not a qualifying path at any grade. You must have experience.

education_substitutable: Education is one way to qualify alongside experience. A candidate with the right degree can qualify even without relevant experience.

education_required: A degree is explicitly required with no experience-only alternative.
